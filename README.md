## About Me

Passionate developer interested in full-stack development, automation, data analysis, and embedded systems. I enjoy fixing problems and providing solutions by designing workable systems.



## Projects

### Website

- [**Personal Page**](https://github.com/CCY-0124/personal-page) – Pixel-art style portfolio with retro game aesthetic and accessibility-focused design
- [**Scholiast**](https://github.com/CCY-0124/Scholiast) – AI-powered web app that turns study materials into interactive quizzes, flashcards, and summaries; tracks learning progress
- [**No Plan No Future**](https://github.com/the-scholiast/NoPlanNoFuture) – Personal productivity and fitness platform: task management, calendar, workout tracking

### Tools

- [**Who's My Sugar Baby**](https://github.com/CCY-0124/who_is_my_sugar_baby) – Discord bot that analyzes receipt text with Gemini AI and calculates bill splits for roommates or friends
- [**Algo Trading**](https://github.com/CCY-0124/Algo_Trading) – Algorithmic trading for cryptocurrency: Glassnode data, backtesting, strategy optimization, live trading, LLM analysis
- [**NPNF Embedded System Add-on**](https://github.com/CCY-0124/NPNF_device) – No Plan No Future embedded system extension
- [**E-job Notification**](https://github.com/CCY-0124/ejobs_notifications) – Fetches BCIT eJOBS (Symplicity) job postings, saves to CSV, posts new listings to Discord
